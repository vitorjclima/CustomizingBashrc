# Customizing .bashrc

Adds more info to your console command line, like:
* Git branch name
* Shows if the active branch is modified
* Timestamp

#### Installing
Copy `.user_prompt` file to your home folder and add `. ~/.user_prompt` to the end of your `.bashrc` file
